Bus Schedule & Reservation Database System

Description:
This project demonstrates a simple SQL database for managing bus routes,
buses, schedules, passengers, and ticket reservations.

Files:
- 01_create_tables.sql : Creates all database tables.
- 02_insert_sample_data.sql : Inserts sample records.
- 03_sample_queries.sql : Example JOIN queries and reports.

Features:
- Route management
- Bus management
- Schedule management
- Passenger records
- Ticket reservations
- Seat availability calculation
- JOIN queries for reporting
